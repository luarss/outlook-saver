# Outlook Saver

Save emails from **Outlook on the web** as `.eml` or `.md` files.

Two ways to use it:

- **[Chrome extension](chrome-extension/)** — one-click save of the email
  you're viewing. No setup; rides your browser session.
- **Tauri app** (`src-tauri/`) — connects over IMAP to auto-save new emails.

See [`chrome-extension/README.md`](chrome-extension/README.md) for install and
usage.
