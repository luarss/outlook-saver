    First-time setup (Azure app registration)
    1. Go to the Azure / Entra portal → App registrations → New registration.
    2. Name it anything; under Supported account types pick the option matching your tenant choice above.
    3. Add a platform → Mobile and desktop applications with redirect URI http://localhost.
    4. Under Authentication, enable Allow public client flows.
    5. Under API permissions, add delegated permission IMAP.AccessAsUser.All (and offline_access, openid, profile).
    6. Copy the Application (client) ID into the field above and Save settings.
